# learning-android-dev

学习：使用 Android Studio 进行安卓开发

Learning: Android development using Android Studio

## 项目说明 (Project Description)

这是一个用于学习 Android 开发的示例项目，包含一个使用 Kotlin 编写的基础 Android 应用。

This is a sample project for learning Android development, containing a basic Android application written in Kotlin.

## 开始使用 (Getting Started)

### 前置要求 (Prerequisites)

- Android Studio (推荐最新稳定版 / Latest stable version recommended)
- JDK 8 或更高版本 (JDK 8 or higher)
- Android SDK API 24-34

### 打开项目 (Opening the Project)

1. 克隆此仓库 (Clone this repository)
   ```bash
   git clone https://github.com/Pectics/learning-android-dev.git
   ```

2. 使用 Android Studio 打开项目 (Open the project with Android Studio)
   - 启动 Android Studio
   - 选择 "Open" 
   - 选择项目根目录

3. 等待 Gradle 同步完成 (Wait for Gradle sync to complete)

### 运行应用 (Running the App)

- 连接 Android 设备或启动模拟器 (Connect an Android device or start an emulator)
- 点击 "Run" 按钮 (▶️) (Click the "Run" button)
